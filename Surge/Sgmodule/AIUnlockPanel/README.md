# AI 服务解锁检测（Surge）

一个仅用于展示出口地区资格的 Surge 面板，检查 ChatGPT、Gemini、Claude、Copilot、Perplexity 与 Grok 的通常地区可用性。

## 安装

在 Surge 中通过下列地址安装模块：

```text
https://raw.githubusercontent.com/PANGPANGDigital/GoodGoodStudyDayDayUp/main/Surge/Sgmodule/AIUnlock/AIUnlockPanel.sgmodule
```

启用后，在 Surge 的**策略组 / Policy Selection**页面会先显示“AI 服务解锁 / 点击卡片开始检测”。点击卡片即可刷新。面板每小时会自动更新一次。

## 结果含义

`✓` 表示出口国家/地区通常受到该服务支持；`×` 表示不在内置地区规则内。它不测试账号登录、订阅、付款资格、实时服务故障或 IP 风控。

地区政策可能变化。ChatGPT、Gemini 和 Claude 的规则分别参考其官方支持地区页面；其余服务只提供保守的地区提示。
